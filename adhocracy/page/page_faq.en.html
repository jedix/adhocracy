Welche unterschiedlichen Dokumente gibt es?

Es gibt einfache Wiki-Seiten, auf denen wie in allen Wikis Inhalte veröffentlicht werden können und Norm-Seiten. Normen können von den Administratoren offiziell angelegt worden sein um über bestimmte Themenbereiche Meinungen einzuholen oder auch verbindliche Abstimmungen einzuholen, oder von Mitgliedern inoffiziell angelegt werden, um unverbindlich unterschiedliche Standpunkte zu bestimmten Themen diskutieren zu können.

Was finde ich unter Dokumente?

Unter Dokumente findet man Norm- und Wiki-Seiten.

Was ist eine Wiki-Seite?

Wiki-Seiten sind wie bei Wikipedia Seiten, auf denen gemeinschaftlich Inhalte erarbeitet werden können

Was ist eine Norm?

Eine Norm grenzt einen thematischen Bereich ein, zu dem Diskussionen und Abstimmungen anhand unterschiedlicher Positionen stattfinden können.

Wie lege ich eine neue Norm unter "Dokumente" an?

Neue Normen können unter "Dokumente" nur von Administratoren angelegt werden. Diese Norm ist dann der "Status Quo" einer Organisation, bis eine andere Variante mehrheitlich gewählt wird

Was ist der "Status Quo" einer Norm

Der "Status Quo" einer Norm ist immer die momentan gültige Regelung innerhalb einer Organisation zu einem bestimmten Thema. In einem Staat können das etwa Gesetze sein, in einer Partei Programmpunkte, in einem Verein oder in einer NGO die Satzung, der Haushaltsplan oder die Strategie.

Wie lege ich eine neue Wiki-Seite an?

Indem ich auf "Dokumente" den Button "Neues Dokument" drücke.

Was ist eine Elternseite?

Wenn ich eine Elternseite beim Erstellen einen neuen Dokumentes, einer neuen Norm oder einer neuen Variante auswähle, wird das neue Dokument automatisch zu einer Unterseite der ausgewählten Elternseite.
