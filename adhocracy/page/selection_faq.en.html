Was ist eine Norm?

Normen sind Regeln, Übereinkünfte oder andere von einer Organisation geteilte Themenbereiche, die zur Diskussion stehen. Zu Normen kann es unterschiedliche Positionen geben, die dann "Varianten" heißen. Normen können in einem Staat etwa Gesetze sein, in einer Partei Programmpunkte, in einem Verein oder in einer NGO die Satzung, der Haushaltsplan oder die Strategie. Normen können aber auch alle anderen Themen sein, zu denen eine Gruppe von Menschen geteilte Standpunkte finden möchten

Was ist eine Variante?

Zu einer Norm kann es unterschiedliche Positionen geben. Die unterschiedlichen Positionen sind somit Varianten einer Norm, die von den Mitgliedern einer Organisation eingebracht und als neue Norm gewählt werden können.

Wie kann man eine neue Variante anlegen?

Unter "Umsetzung" in einem Vorschlag drückt man bei dem gewünschten Absatz auf "auswählen und diskutieren". Auf der Seite auf die man dann gelangt sieht man alle bereits eingebrachten Varianten. Entspricht keine der Vorhandenen den eigenen Vorstellungen kann man durch Drücken von "neue Variante" eine neue anlegen.

Wie kann man eine neue Norm innerhalb eines Vorschlags anlegen?

Möchte man unter "Umsetzung" einen neuen Punkt einbringen, den als als relevant für das Erreichen des Ziels des Vorschlags erachtet, kann man durch Drücken auf "Umsetzung ergänzen" eine neue Norm anlegen.

Für welche Variante einer Norm steht ein Vorschlag?

Immer für die Variante, die von der Seite des jeweiligen Vorschlags aus am häufigsten unterstützt wird.

Was ist der "Status Quo" einer Norm

Der "Status Quo" einer Norm ist immer die momentan gültige Regelung innerhalb einer Organisation zu einem bestimmten Thema. In einem Staat können das etwa Gesetze sein, in einer Partei Programmpunkte, in einem Verein oder in einer NGO die Satzung, der Haushaltsplan oder die Strategie.

Was ist eine Elternseite?

Wenn ich eine Elternseite beim Erstellen einen neuen Dokumentes, einer neuen Norm oder einer neuen Variante auswähle, wird das neue Dokument automatisch zu einer Unterseite der ausgewählten Elternseite.
