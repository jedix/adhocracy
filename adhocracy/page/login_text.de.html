Zum Anmelden oben rechts auf „Registrieren“ klicken.


Nun die Felder unter "Registrieren" ausfüllen und unten den Button "Registrieren" klicken.


Jetzt Sind Sie angemeldet. Herzlich Willkommen!
Um Profil-Einstellungen vorzunehmen bitte „Einstellungen“ oben rechts klicken.


Hier können einige individuelle Einstellungen vorgenommen werden:

  1. (a) Hier können Sie Ihren Nutzernamen ändern. (Vorsicht: Zum anmelden benötigen sie weiterhin den Nutzernamen, den Sie bei der Anmeldung angegeben haben.)
  2. (b) Hier können Sie ihr Passwort ändern
  3. (c) Hier können Sie Ihre E-Mail-Adresse ändern
  4. (d) Um Mitteilungen zu erhalten klicken Sie "Aktivierungs-Link versenden". Bestätigen Sie im Anschluss die E-Mail, die Sie erhalten.
  5. (e) In diesem Feld können Sie Inhalte für Ihre Profilseite eingeben. Formatierungshilfen finden Sie unten rechts.
  6. (f) Hier können Sie einstellen, wie lang die Listen in Adhocracy für Sie angezeigt werden.
  7. (g) Wenn Sie "speichern" klicken, werden alle Ihre Änderungen übernommen.

Um in Projektgruppen der Enquete Kommission Internet und digitale Gesellschaft als 18. Sachverständiger oder 18. Sachverständige mitarbeiten zu können, klicken Sie auf "Projektgruppen."
Hier finden Sie eine Übersicht aller momentan arbeitenden Projektgruppen.


Wählen Sie per Anklicken die Projektgruppe oder Arbeitsgruppe aus, in der Sie mitwirken möchten.


Klicken Sie "beitreten" um Mitglied dieser Gruppe zu werden. Sie haben jetzt die Möglichkeit aktiv mitzuarbeiten und an Abstimmungen teilzunehmen. Möchten Sie weiteren Gruppen beitreten, klicken Sie erneut "Projektgruppen" oben rechts und wiederholen Sie den Vorgang.
